# SmartHire AI

**AI-Powered Placement Preparation Platform**  
*Team TwinTech — Yashvant & Shivani*

## Modules

| # | Module | Description |
|---|--------|-------------|
| 1 | User Management | Registration, Auth, Profile, Roles (FR-1) |
| 2 | AI Mock Interview | Adaptive NLP interviews with Gemini AI (FR-2) |
| 3 | Resume Analyzer | ATS scoring, keyword analysis, optimization (FR-3) |
| 4 | Smart Job Tracker | Kanban board, deadlines, calendar sync (FR-4) |
| 5 | Career Recommendation | Career paths, readiness prediction (FR-5) |
| 6 | Analytics & Insights | 25 analysis types with predictive scoring (FR-6) |
| 7 | Notifications | Multi-channel smart alerts (FR-7) |
| 8 | Voice Intelligence | Speech-to-text, pronunciation, fluency (FR-8) |
| 9 | TPO Module | Placement officer dashboards & reports (FR-9) |
| 10 | Admin Module | System management & audit logs (FR-10) |
| 11 | Recruiter Module | Job posting, ATS screening, hiring pipeline (FR-11) |
| 12 | Integration Module | Gemini, LinkedIn, Calendar, GitHub (FR-12) |

## Tech Stack

- **Frontend:** React 18 + Vite + Tailwind CSS + Redux Toolkit
- **Backend:** Node.js + Express.js + Socket.io
- **AI/ML:** Google Gemini Pro + spaCy + LangChain.js
- **Database:** MongoDB Atlas + Redis
- **Cloud:** AWS S3 + Cloudinary
- **DevOps:** Docker + GitHub Actions

## Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```
